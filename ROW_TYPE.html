16. Write a PL/SQL block to update salary using %ROWTYPE variable.

    v_emp_data SCOTT.EMP%ROWTYPE;
BEGIN
    SELECT * INTO v_emp_data FROM SCOTT.EMP WHERE EMPNO = 7844;
    UPDATE SCOTT.EMP SET SAL = v_emp_data.SAL + 100 WHERE EMPNO = v_emp_data.EMPNO;
    DBMS_OUTPUT.PUT_LINE('Updated salary for ' || v_emp_data.ENAME);
    COMMIT;
END;

Message
Updated salary for TURNER
